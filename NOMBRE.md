*Parte del proyecto Séptima - Aldraverse*


Implementa un 'Modo Laboratorio Seguro' dentro del ecosistema educativo biomédico. Este modo debe permitir a los estudiantes modificar libremente todos los parámetros de simulación (fisiológicos, de señales, de microcontroladores) y experimentar con el código sin afectar su progreso guardado o los datasets base. Debe incluir una opción para 'restablecer a valores predeterminados' o 'volver al estado anterior' fácilmente. Este entorno debe ser ideal para la práctica deliberada y la exploración sin consecuencias.

Desarrolla un módulo de 'Recorrido Guiado' para el proyecto biomédico. Este módulo debe guiar a los estudiantes a través de cada sistema (empezando por el cardiovascular) en una secuencia definida: anatomía 3D interactiva, simulación fisiológica con parámetros ajustables, visualización de señales biomédicas (ECG) y una introducción básica a la programación de microcontroladores para interactuar con estas señales. Cada paso debe incluir explicaciones claras y concisas, así como pequeños retos o preguntas para reforzar el aprendizaje.
